Esta aplicación esta basada en el Stack de tecnologías MEVN, para el diseño se utiliza el framework bootstrap
Esta aplicacion Web consiste en un administrador de tareas el cual puede programar tareas especiales de acuerdo a las necesidades solicitadas.
 
Ejecutar servidor: npm run dev
Ejecutar Mongo: mongod

Requsitos necesarios:
	1. instalar Node.js(npm install)
	2. Instalación Express(install express)
	3. Instalación Vue(install vue)
	4. Instalación CLI de Vue(npm install -g @vue/cli)  
	5.Instalar Nodemoon (npm install nodemon) // refrescar el servidor automáticamente para ver los cambios
		se debe crear los scripts en package.json, para este caso ya están creados.

	6. Crear Middleware(funciones): Instalar morgan (npm install morgan) //visualizar las peticiones que envia el navetgador
	7 Intalación Mongoose (npm install mongoose)

	8. ejecutar mongod
	9. ejecutar servidor dev
	10. ejecutar webpack
