Esta aplicación esta basada en el Stack de tecnologías MEVN y tambien para el diseño utiliza el framework de bootstrap

Ejecutar servidor: npm run dev
Ejecutar servidor de Mongo (mongod en el cmd)

Requsitos necesarios:
	1. instalar Node.js(npm install)
	2. Instalación Express(install express)
	3. Instalación Vue(install vue)
	4. Instalación CLI de Vue(npm install -g @vue/cli)  //ingresar comandos por consola
	5.Instalar Nodemoon (npm install nodemon) // refrescarse el servidor automaticamente para ver los cambios
		se debe crear los scripts en package.json, para este caso ya estan creados.

	6. Crear Middleware(funciones): instalar morgan (npm install morgan) //visualizar las peticiones que envia el navetgador
	7 Intalacion Mongoose (npm install mongoose)

	8. ejecutar mongod
	9. ejecutar servidor dev
	10. ejecutar webpack
